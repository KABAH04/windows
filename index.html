Actualite du dans le domaine de la programmation

MICROSOFT BUILD

Microsoft Build est l’événement annuel de Microsoft destiné aux développeurs. L’occasion pour la firme de présenter ses nouvelles mises à jour logicielles, qui permettent de coder plus facilement, et proposer de nouvelles applications et services sur Microsoft 365, Windows, Power Platform ou encore Azure.

Microsoft 365

À l’occasion de sa conférence Microsoft Ignite, à destination des développeurs et des professionnels, la firme a dévoilé une nouvelle application visant à faciliter le travail hybride.

Microsoft Loop

Dans un post de blog, la firme a dévoilé Microsoft Loop qui s’inscrit comme la grande nouveauté de la suite Office. Cette nouvelle application combine un canevas flexible avec des composants dynamiques qui se déplacent librement et restent synchronisés entre les applications. L’objectif : permettre aux équipes de réfléchir, planifier et créer ensemble, et partout.

Microsoft Teams

Des outils pour améliorer le travail collaboratif : une nouvelle expérience grâce à Live Share, va permettre aux participants d’interagir avec les applications partagées, directement depuis la fenêtre de réunion Teams. Des outils pour intégrer des fonctionnalités de collaboration Teams dans d’autres applications : une option de contrôle de collaboration sera disponible dans Microsoft Power Apps, sa suite d’applications fournissant un environnement de développement rapide pour créer des applications métiers personnalisées. Cette fonctionnalité, dont la preview sera disponible cet été, permet d’intégrer des options telles que le chat, les réunions, les tâches ou les partages de fichiers dans Power Apps. De nouvelles API sont prévues pour Microsoft Graph, disponible en preview. Elles permettent aux développeurs d’intégrer des fonctionnalités et des contenus Teams dans leurs propres applications. Azure Communication Services, ensemble d’API de communication, vidéo et SMS, permet aux développeurs de générer et de déployer des applications pour des visites virtuelles en quelques minutes seulement sans codage. Déploiement de l’extension de kit de ressources Teams pour Visual Studio Code et Visual Studio : Microsoft Teams Toolkit permet aux développeurs de créer, générer, déboguer et publier des applications pour Teams sur des appareils web et mobile en quelques minutes. Des mises à jour dans l’App Source de Microsoft favorisent l’engagement des utilisateurs dans Teams : la firme annonce des recommandations intelligentes pour permettre aux développeurs de maximiser l’engagement des utilisateurs, et ainsi mieux monétiser leurs applications dans le Teams App Store, avec plus de possibilités. Les API pour configurer les achats intégrés sont également disponibles, pour faciliter le passage des versions gratuites vers les abonnements payants dans Teams. Un outil pour vérifier automatiquement la conformité des applications à Microsoft 365 Des nouvelles applications collaboratives pour Teams en partenariat avec les éditeurs indépendants :Bosch Cognitive Service Maintenance, Figma, MURAL, Observable, SAP S/4HANA ou encore ServiceDesk Plus Cloud.

Power Platform

Power Platform est la gamme d’applications de Microsoft qui regroupe des logiciels de business intelligence, mais aussi de développement et de connectivité d’applications. Power Pages, un outil pour construire des sites web professionnels avec très peu de code : cet outil sera désormais un logiciel autonome de la famille Power Platform et de nouvelles fonctionnalités sont disponibles pour améliorer l’expérience utilisateur. Un outil pour convertir rapidement les fichiers en une application Power Apps :Il est maintenant possible de passer d’un simple fichier (conception Figma, PDF, PowerPoint, formulaire papier, capture d’écran…) à une application Power Apps fonctionnelle avec une interface utilisateur et un schéma de données. Power Virtual Agent pour créer des bots intelligents : cet outil permet de combiner les fonctionnalités d’Azure Bot Framework avec un système low code, pour simplifier le développement des bots intelligents. Datamart, une nouvelle fonctionnalité de Power BI : cet outil de Power BI Premium permet aux utilisateurs de découvrir des informations utiles et exploitables à partir de leurs propres ensembles de données dépendantes et indépendantes. L’automatisation des outils pour le Cloud dans Power Automate : le logiciel de Microsoft, qui permet de rationaliser les tâches répétitives et d’automatiser les processus, va désormais permettre d’exécuter les fonctionnalités sur des machines virtuelles alimentées par Azure.

Windows

Microsoft a profité de son événement annuel pour annoncer de nouvelles fonctionnalités destinées à son système d’exploitation, ainsi qu’au Microsoft Store.

Une amélioration des outils de développement d’applications pour améliorer du code existant sur Windows : pour faciliter la création d’applications multiplateformes (Windows, Android, Linux, .net, .web, C++, etc.), la firme améliore Windows App SDK et WinUI 3. Des nouvelles opportunités pour atteindre les clients via le Microsoft Store : il est désormais possible de choisir parmi le moteur de commerce natif proposé par Microsoft, qui impose un partage des bénéfices, ou un moteur de commerce personnalisé au sein de l’application, qui permet de conserver l’ensemble des revenus. La suppression du programme de liste d’attente pour les applications Win32 dans le Microsoft Store est nécessaire. Un nouveau kit de développement qui offre des fonctionnalités d’intelligence artificielle : Microsoft annonce son projet Volterra, une boîte à outil pour Windows, qui permet d’explorer de nombreux scénarios d’IA dans le cadre du développement d’applications.

Azure
Le service de cloud computing de Microsoft a lui aussi bénéficié de nombreuses annonces.
Azure AI

Azure Cognitive Services : une mise à jour pour aider les développeurs à déployer des API de haute qualité, et intégrer à leurs applications plus de fonctionnalités. Apprentissage automatique : pour aider les développeurs à accélérer le déploiement de modèles basés sur l’IA. Applied AI : plusieurs nouveautés sur Azure Form Recognizer, Azure Bot Service, Power Virtual Agents, ainsi que Azure Metrics Advisor.

Azure Cloud Native & App Platform

Azure Container Apps : déploiement d’outils conçus pour exécuter des micro-services et des applications conteneurisées sur une plateforme sans serveur. Azure Communication Services : mise à jour de la fonctionnalité, avec l’arrivée de Azure Communication Services Mobile UI Library. Azure Spring Apps Azure Kubernetes Service : amélioration de l’expérience pour les développeurs. Azure Portal : Service Bus Explorer Azure API Management : mise en place d’une plateforme de gestion hybride et multi-cloud pour les API. Azure App Service : mise à jour autour du service Google Remote Procedure Call Support, et lancement de l’application Service Landing Zone Accelerator pour aider les clients à effectuer un déploiement avec Azure App.
